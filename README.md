# Void Contrast

Dark High Contrast with the noise stripped out. Pure black, zero clutter.

Built from VS Code's Dark High Contrast base — borders removed, pure black background, syntax highlighting preserved.

## Install

1. Download [`void-contrast-0.1.0.vsix`](https://github.com/jaksm/vscode-void-contrast/raw/main/void-contrast-0.1.0.vsix)
2. In VS Code: `Cmd+Shift+P` → **Extensions: Install from VSIX** → select the file
3. `Cmd+K Cmd+T` → select **Void Contrast**

Or from terminal:

```bash
curl -Lo /tmp/void-contrast.vsix https://github.com/jaksm/vscode-void-contrast/raw/main/void-contrast-0.1.0.vsix
code --install-extension /tmp/void-contrast.vsix
```

## Uninstall

`Cmd+Shift+P` → **Extensions: Uninstall** → Void Contrast
